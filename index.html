Introducción

La aplicación Map Tags nos enseña cómo integrar Google Maps en nuestra aplicación. La API de Google Maps y la librería jquery-ui-map facilitan la edición de un mapa con marcadores (markers) y figuras (overlays), las cuales podemos personalizar y almacenar, utilizando para ello una base de datos MongoDB.

Instalación

Puedes clonar el repositorio de esta aplicación utilizando Git.

$ mkdir maptags
$ cd maptags
$ git clone git://github.com/danielsd10/demo.maptags.git

También puedes descargar directamente.

Tecnologías aplicadas

  • PHP como lenguaje backend
  • MongoDB como gestor de base de datos
  • Bootstrap
  • jQuery
  • Google Maps API
  • jquery-ui-map

Funcionalidades incluidas

  • Visualización de etiquetas por tipo
  • Herramientas de registro
  • Formulario de registro
  • Almacenamiento en base de datos
  • Visualizar detalles de una etiqueta
